# CDL Platform - QA Automation Framework

This repository contains the end-to-end automated test suite for the Container Don Land (CDL) platform. The framework is built using **Playwright** and **JavaScript** to validate core business logic, user interface workflows, and prevent regressions.

## 🚀 Features & Scope

*   **Containers Dashboard Testing:** Automates the verification of data filters, layout rendering, and pagination elements to optimize workflows for power users.
*   **Top Up Credit Modal Verification:** Validates logical guardrails on financial workflows, ensuring fields default correctly and form submissions are blocked until valid input is given.
*   **Data-Driven Capabilities:** Engineered to seamlessly cycle through dynamic test cases directly sourced from project spreadsheets (`.xlsx`).
